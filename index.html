My Projects

Cyber Bullying Tweets Detection using Machine Learning

A machine learning project that detects cyberbullying tweets by analyzing text patterns and applying classification algorithms to identify offensive or harmful content on social platforms. This study introduces a machine learning-based tweet classifier.
* It aims to automatically categorize tweets into predefined categories.
* The system analyzes tweet content based on semantic meaning and context.
* Preprocessing steps include tokenization, stemming, and stopword removal.
* Feature extraction prepares the text data for classification.
* Models such as Naive Bayes, SVM, and CNN are trained on labeled datasets.
* These models learn patterns linking tweet content to categories.
* Evaluation metrics include accuracy, precision, recall, and F1-score.
* Results show the classifier is robust and effective.
* This tool supports sentiment analysis, trend detection, and social media insights.
